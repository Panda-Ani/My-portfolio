Sudoku Game

Using Python

Python is a potent language that is used for a variety of works. Some significant works include machine learning, web development, game development, etc. Here I have made a simple Sudoku game using Python and given it a minimalistic GUI using the Pygame library.

A typical game of Sudoku Contains 9 rows and 9 columns, and the 9×9 matrix is divided further into a 3×3 matrix. The rule of the game is that no two same numbers can be on the same row or column or the same 3×3 box.

Here I have used a bunch of functions to make the GUI, such as to take input, display the board, erase the previous answer, and Made a timer to show the user the time taken by them. The board will be filled automatically if you click an empty place and enter space. It uses the Backtracking Algorithm to solve the problem. Which is better than the brute force recursive algorithm as it rejects the unnecessary steps to check 981 possible solutions in brute force.
