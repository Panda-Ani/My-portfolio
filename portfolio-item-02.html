Student Database Management System

Using Cpp

Object-Oriented Programming (OOP) is the heart of Cpp and can be used to make effective programs in Cpp. Here I have illustrated that by making a simple student database management system model purely using Cpp.

The program takes the input of necessary academic information of the student and calculates whether they are eligible for application or not, then registers the student. It divides the student into two streams, JEE and NEET. According to the student database. Then you can use the program to display each student's information, And the data gets stored in a file. So that even if the program closes, the data is stored in that file, and to make it possible, I have used File Handling, another Interesting Feature of Cpp.
